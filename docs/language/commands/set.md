# Command: Set
Sets the target register to the information provided as a value.
```fiber
set {target} {value}
```

## Valid Combinations
| Target | Value |
|:-:|:-:|
| Register Value | Register Value |
| • | Register Address |
| • | Float |
| • | Int |
| • | Uint |
| • | Hex |
| Register Address | Register Address |
| • | Register Value |
| • | Local |
| • | Parse |
